Online Fibernet Billing System

The Web-based Fibernet Billing and Payment System is a platform that digitalizes subscription and billing processes for Internet service providers. Admins can manage users, assign plans, and track payments from a dashboard, while customers can log in to view their plan details, download invoices, and check payment history. The system automatically generates invoices based on the billing cycle, reducing manual effort and improving transparency between provider and customer.

Smart Traffic Light Simulator

This project simulates an intelligent traffic light controller using microprocessor programming, where the green signal duration changes dynamically based on vehicle count. It demonstrates how real-world traffic automation can be implemented using software logic without hardware sensors. The model focuses on managing a single traffic junction but can later be scaled to multi-lane and multi-junction systems. It also provides a foundation for future integration with sensors and embedded devices for real-time traffic control.
